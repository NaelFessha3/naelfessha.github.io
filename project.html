Discovery Project — Temperature-Controlled Fan System

Fall 2024 Embedded Systems View Code

The Problem

For my Discovery Project, I challenged myself with a hands-on system involving real ECE concepts — sensors, microcontroller programming, and motor control. The question I set out to answer was: "How can I build a system that senses temperature and automatically controls a fan?"

The Process

1. Empathize & Understand

I researched temperature sensors and learned that the DHT11 would be easy to interface with an Arduino. I also studied how PWM works for controlling fan speed and how motor drivers protect the microcontroller.

2. Define

The system needed to:

  • Read live temperature values
  • Display readings clearly on an LCD
  • Turn a fan on when temperature rises
  • Adjust fan speed automatically using PWM

3. Ideate — Components Selected

  • Arduino Uno
  • DHT11 Sensor
  • 16×2 LCD Display
  • L298N Motor Driver
  • 5V DC Fan

I planned wiring diagrams, reviewed data sheets, and mapped each component's pin connections.

4. Prototype

  • Connected the DHT11 to Arduino pin 7
  • Wired the LCD using 4-bit parallel mode
  • Connected the fan to the L298N motor driver outputs
  • Programmed PWM logic to modulate speed based on temperature thresholds

5. Test

I tested the system by blowing warm air on the sensor and observing the LCD data and fan behavior. I also tested multiple fan types and learned how current limitations affect performance.

The Outcome

The final system successfully reads real-time temperature, displays it on the LCD, and controls fan speed automatically — a working closed-loop control system using microcontroller logic, sensors, and output hardware.

Successes & Challenges

Successes

  • Accurate temperature readings from the DHT11
  • Real-time display updating on the LCD
  • Automatic fan speed control via PWM
  • Successful integration of all hardware components

Challenges

  • Voltage drops caused the LCD to dim when the fan activated
  • A larger 5V fan required too much current for the Arduino
  • DHT11 sensor errors until timing delays were tuned
  • L298N voltage drop reduced fan performance

Skills Gained

  • Arduino / C++
  • DHT11 Integration
  • LCD Interfacing
  • PWM Motor Control
  • Power Management
  • Circuit Debugging
  • System Integration
